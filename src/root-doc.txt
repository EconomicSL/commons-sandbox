==Overview==
Our API consists of several different layers each of which is represented by its own package.

===`acl` package===
Our high-level actor communication language is influenced by, but not slave to, the
[[http://www.fipa.org/ Foundation for Intelligent Physical Agents (FIPA)]]
[[http://www.fipa.org/specs/fipa00037/SC00037J.pdf Agent Communication Language (ACL)]].

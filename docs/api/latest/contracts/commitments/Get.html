Get

Related Doc: package commitments

class Get extends Contract

A Contract that, when acquired, acquires the underlying contract at the contract horizon (i.e., at the last possible moment) regardless of when the composite contract Get(contract) was acquired.

Note

If you acquire Get(contract), then you must acquire contract at the contract expiry date. The composite contract Get(contract) expires at the same moment that contract expires.

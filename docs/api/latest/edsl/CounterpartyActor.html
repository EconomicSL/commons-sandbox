trait CounterpartyActor extends AnyRef

Trait defining behavior for an actor that is a counterparty to the underlying Commitment of some ContractActorLike.

Note

    The act of breaking the terms of some existing Commitment.

    A edsl.ContractActorLike actor whose underlying Commitment the Counterparty actor wishes to break.

    Note

    When a Counterparty actor breaks the terms of some existing Commitment it becomes in "breach of contract." The Counterparty actor sending ContractBroken is informing the receiver that it no longer has the intention of performing the actions specified in the receiver's underlying Commitment.

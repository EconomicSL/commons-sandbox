InformRef

Related Doc: package acts

case class InformRef[D](conversationId: UUID, descriptor: (D) ⇒ Boolean) extends CommunicativeAct with Product with Serializable

A message sent from some CommunicatingActor (i.e., sender) to another CommunicatingActor (i.e., receiver) informing the receiver of object(s) which satisfy some descriptor.

D

is the type of objects described by the descriptor.

conversationId

is an expression used to identify a sequence of communicative acts that together form a conversation.

descriptor

is a function describing some required characteristics of the object.

Note

The InformRef message is sent by the sender using the informRef action.

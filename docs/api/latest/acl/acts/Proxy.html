Proxy

Related Doc: package acts

case class Proxy[D](conversationId: UUID, message: CommunicativeAct, descriptor: (D) ⇒ Boolean, constraint: (Beliefs) ⇒ Boolean) extends CommunicativeAct with Product with Serializable

A message sent from some CommunicatingActor (i.e., sender) to another CommunicatingActor (i.e., receiver) in order to proxy some embedded message.

conversationId

is an expression used to identify a sequence of communicative acts that together form a conversation.

message

is the embedded CommunicativeAct which is being proxied.

descriptor

is a proposition denoting a collection of actors to whom the Proxy message should be sent by the receiver.

constraint

is a proposition describing a termination condition for the propagation of the message.

Note

The Proxy message is sent by the sender using the proxy action.
